Organize the Web: Constructing Domains Enhances Pre-Training Data Curation

Alexander Wettig Kyle Lo Sewon Min Hannaneh Hajishirzi Danqi Chen Luca Soldaini
Paper Hugging Face Code

Overview

Modern language models are trained on vast web-crawled datasets, but their unstructured nature makes systematic data curation challenging. We introduce WebOrganizer, a framework that brings structure to web-based pre-training data through comprehensive domain classification, focusing on both topic and format domains.

Key Contributions

  • Domain Taxonomies: Development of comprehensive taxonomies for categorizing web content by topic and format
  • Efficient Classification: Distillation of large language model annotations into lightweight, efficient domain classifiers
  • Domain Mixing: Novel approaches for combining content from different domains to improve downstream task performance
  • Quality Integration: Methods for combining domain-based and quality-based data curation approaches

Key Findings

  • Combining insights about effective topics and formats leads to improved model performance
  • Domain mixing enhances existing quality-based data selection methods
  • Quality-based methods implicitly affect domain distribution in predictable ways
  • Domain-aware curation provides a valuable complement to quality-based approaches
